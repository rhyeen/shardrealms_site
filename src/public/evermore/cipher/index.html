Evermore Cipher


If the message is encoded with the cipher symbols, use the keyboard below to type the decoded letter into the input above.

If the message is encoded with the vampyric letters, use the keyboard below to type the decoded letter into the input above.


Letters can be entered as-is, then "Decode Letters". Numbers should be separated by spaces, then "Decode Numbers".

You can mix letters/symbols and numbers (no spaces) if they are mixed in one word, then "Decode Combination".

For example, try:

  • Input: "jgnnq yqtnf", then click "Decode Letters" and look at result #25
  • Input: "17 15 18 2", then click "Decode Numbers" and look at result #4
