How to contribute to Shardrealms

Thanks for your interest in contributing to Shardrealms. It’s been a lot of effort on our end to get the project to where it is today, but we still have a long ways left to go! That’s why we really appreciate your willingness to help.

We want this to be an enjoyable project for you. So if you ever find yourself unsure of where to start or what to do, feel free to reach out to our admins at shardrealms@gmail.com. That being said, we’re all extremely busy with our daytime jobs, families, and the development of Shardrealms. So if we don’t respond in some time, please be patient.

We want to be sure to provide you with the resources you need to contribute without our guidance as much as possible. We will be using some tools and documents to lead you to approved and confident contributions. Here are a few things to help get you started.

Slack team

We will be using Slack to communicate all things regarding Shardrealms. The quickest and easiest way to get answers to all your questions and to join together with fellow contributors is to post on the Slack channels. To start, join the Slack team.

Shardrealms App

The Shardrealms Creation App is a powerful open-source tool for a game master to keep track of his campaign story. It will eventually be available to the public, but for now, it is only accessible by running the application on your computer. Basic web software development skills are a requirement to contribute to the application.

If you want to contribute to the development of the Shardrealms App, you can find the Shardrealms App Base Repository on Github with instructions on how to get started on contributions.

Shardrealms Universe

The Shardrealms Universe is a high-fantasy story setting for tabletop RPGs. It is entirely free, community-driven, and compatible with the Shardrealms System and App. Contributors can be just about anyone: story writers, proofreaders, testers, artists, designers, etc.

If you want to contribute to the development of the Shardrealms Universe, check out the Introductory Shardrealms Universe document.

Shardrealms System

The Shardrealms System is a high-fantasy tabletop RPG system, similar to D&D or Pathfinder. It is entirely free, community-driven, and compatible with the Shardrealms Universe and App. Contributors can be just about anyone: play-testers, content creators, proofreaders, graphic designers, etc.

If you want to contribute to the development of the Shardrealms System, check out the Introductory Shardrealms Systems document.

Thanks again for joining the team, and we look forward to working with you!
